# 🐍 Python Journey

## 👩‍💻 Giới thiệu

Xin chào, mình là **Nguyễn Thị Tuyết Nhi**.  
Đây là dự án học Python của mình được xây dựng nhằm ghi lại quá trình học tập, thực hành và phát triển kỹ năng lập trình Python từ cơ bản đến nâng cao.

Mục tiêu của dự án:

- Luyện tập Python mỗi ngày
- Thực hành qua bài tập và mini project
- Làm quen với GitHub và quy trình làm việc nhóm
- Xây dựng nền tảng cho lập trình và phát triển phần mềm
